# Clean Store Bot v3

Telegram магазин-бот на aiogram 2.25.1 + PostgreSQL + Railway.

## Возможности
- /start
- каталог по категориям
- корзина
- покупка с проверкой баланса
- личный баланс
- мои заказы
- /admin
- авторизация по логину и паролю
- добавление категорий
- добавление товаров
- изменение цены
- пополнение баланса
- просмотр товаров и заказов

## Railway
1. Загрузи проект в GitHub.
2. Подключи репозиторий в Railway.
3. Добавь PostgreSQL.
4. Укажи переменные из `.env.example`.
5. Deploy.

## Обязательные переменные
- BOT_TOKEN
- DATABASE_URL
- ADMINS
- ADMIN_LOGIN
- ADMIN_PASSWORD
- SHOP_TITLE
